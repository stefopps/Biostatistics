Phase 1 — The Foundation

Data types • Normal distribution • Central tendency
Data Types Distributions
Mnemonic: NOIR
  • Nominal — Categories with no order (e.g., blood type, sex).
  • Ordinal — Ordered categories, unequal intervals (e.g., pain scale 1–10).
  • Interval — Equal intervals, no true zero (e.g., temperature °C).
  • Ratio — Equal intervals + true zero (e.g., weight, height).
  • 68-95-99.7 Rule — 1 SD covers 68 %, 2 SD covers 95 %, 3 SD covers 99.7 % of data.
  • Mean vs. Median — Skewed data: mean is pulled toward the tail; use median for skewed distributions.
  • Positive skew — Mean > Median > Mode (tail to the right).
  • Negative skew — Mean < Median < Mode (tail to the left).

Phase 2 — Observational Study Designs

Case-control • Cohort • Cross-sectional • Ecological
Study Design Epidemiology
  • Case-Control — "Rare Disease Detective." Start with disease, look backward for exposure. Measure: Odds Ratio (OR).
  • Cohort (Prospective) — Start with exposure, follow forward for outcome. Measure: Relative Risk (RR).
  • Cohort (Retrospective) — Both exposure and outcome already occurred; look backward using records. Measure: RR.
  • Cross-Sectional — Snapshot in time; exposure and outcome measured simultaneously. Measure: Prevalence.
  • Ecological — Population-level data; cannot infer individual risk (ecological fallacy).
  • RCT — Gold standard; random allocation eliminates confounding.
RR = [a/(a+b)] ÷ [c/(c+d)]
OR = (a×d) ÷ (b×c)
RR ≈ OR when disease is rare (<10 %)

Phase 3 — Diagnostic Tests & Errors

Sensitivity • Specificity • PPV • NPV • Type I/II errors
Diagnostics Hypothesis Testing
Mnemonic: SnNout / SpPin
Sensitivity = TP / (TP + FN) — "Rules OUT disease when negative"
Specificity = TN / (TN + FP) — "Rules IN disease when positive"
PPV = TP / (TP + FP) — depends on prevalence
NPV = TN / (TN + FN) — depends on prevalence
LR+ = Sensitivity / (1 − Specificity)
LR− = (1 − Sensitivity) / Specificity
  • Type I Error (α) — False positive; rejecting H₀ when it is true. Controlled by significance level (p-value threshold).
  • Type II Error (β) — False negative; failing to reject H₀ when it is false.
  • Power (1 − β) — Probability of correctly detecting a true effect. Increases with larger N, larger effect size, or higher α.
  • ROC Curve — Plots sensitivity vs. (1 − specificity). Best test = top-left corner. AUC = 1.0 is perfect.
  • Prevalence & PPV — High prevalence → higher PPV; low prevalence → lower PPV (even with good Sp).

Phase 4 — Bias, Confounding & Validity

Selection bias • Information bias • Confounding • Internal vs. external validity
Bias Validity
  • Selection Bias — Systematic error in how participants are chosen (e.g., Berkson's bias, healthy worker effect).
  • Recall Bias — Cases remember exposures better than controls (common in case-control studies).
  • Lead-time Bias — Screening detects disease earlier; survival appears longer without true benefit.
  • Length-time Bias — Screening preferentially detects slow-growing (less lethal) disease.
  • Confounding — A third variable associated with both exposure and outcome (e.g., smoking confounds alcohol–lung cancer link). Fix: randomization, restriction, matching, multivariate analysis.
  • Effect Modification — Effect of exposure differs across strata of a third variable; NOT a bias — report separately.
  • Internal Validity — Results are correct for the study population.
  • External Validity (Generalizability) — Results apply to other populations.

Phase 5 — Advanced Statistical Tests

ANOVA • Chi-square • Correlation • Regression • Forest plots
Inferential Stats Meta-analysis
  • t-test — Compares means of 2 groups (parametric; assumes normal distribution).
  • ANOVA — Compares means of 3+ groups; follow up with post-hoc test to find which groups differ.
  • Chi-square (χ²) — Compares proportions / categorical data between groups.
  • Pearson r — Linear relationship between 2 continuous variables (−1 to +1). r² = variance explained.
  • Spearman ρ — Non-parametric rank correlation; use when data are ordinal or non-normal.
  • Regression — Linear: predicts continuous outcome. Logistic: predicts binary outcome (outputs OR).
  • Forest Plot — Meta-analysis summary. Each row = one study; diamond = pooled estimate. If CI crosses 1.0 → not significant.
  • Funnel Plot — Checks for publication bias in meta-analysis; asymmetry suggests bias.
p-value < α → reject H₀ (statistically significant)
95 % CI that excludes 1.0 (for RR/OR) or 0 (for difference) → significant

Phase 6 — Prevention & Screening

Primary • Secondary • Tertiary • NNT • NNH
Prevention Clinical Application
  • Primary Prevention — Prevent disease before it occurs (e.g., vaccination, lifestyle counseling).
  • Secondary Prevention — Early detection & treatment (e.g., mammography, Pap smear).
  • Tertiary Prevention — Reduce disability from established disease (e.g., cardiac rehab).
  • Absolute Risk Reduction (ARR) — Riskcontrol − Risktreatment.
  • Relative Risk Reduction (RRR) — ARR ÷ Riskcontrol.
  • NNT — 1 / ARR. Smaller = more effective treatment.
  • NNH — 1 / Absolute Risk Increase. Larger = safer drug.
ARR = CER − EER
NNT = 1 / ARR    NNH = 1 / ARI
